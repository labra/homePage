XML - Lenguajes y estándares para la Web

Profesor: Jose Emilio Labra Gayo

Material sobre XML de la asignatura Lenguajes y estándares para la Web en el Máster Oficial de Ingeniería Web de la Universidad de Oviedo.

Objetivos

Los objetivos de esta parte de la asignatura son:

  • Conocer la sintaxis del lenguaje XML
  • Conocer las tecnologías de validación de documentos XML: DTDs y XML Schema
  • Crear vocabularios XML y validar dichos vocabularios mediante XML Schema
  • Conocer la sintaxis XML Namespaces
  • Aplicar tecnologías de transformación de documentos XML: XSLT y XPath

Evaluación

La evaluación de esta parte se basará en que cada alumno realice los siguientes ejercicios. Pueden obtenerse en formato HTML o PDF

Para superar la materia deben realizarse todos los ejercicios, salvo los que se indican con la palabra Opcional

Los ejercicios opcionales se realizarán para subir nota

Formato de entrega

La entrega se realizará a través del Campus Virtual y de la página Web de cada alumno.

En el campus virtual se incluirá:

  1. El código fuente comprimido (ficheros XML, XSD y XSL)
  2. Un enlace a la IP con el resultado final (ficheros HTML o en otros formatos de salida generados)
  3. Un pequeño documento (¡Máximo una hoja!) que explique brevemente los ejercicios realizados o alguna otra circunstancia que deba tenerse en cuenta para la corrección.
