Ejercicios para Jose E. Labra (Sept. 2009)

Web Semántica y Nuevos Avances en Web Semántica

Asignaturas del Máster Oficial de Ingeniería Web de la Universidad de Oviedo.

  1. Crear una consulta SPARQL sobre datos en la DBPedia que permita obtener información interesante sobre un determinado tema. Se valorará la combinación entre datos de diferentes fuentes, así como la originalidad de la consulta. Explicar los resultados que se obtengan.
  2. Resolver mediante OWL el problema de Einstein
  3. El último ejercicio depende del tipo de rama de cada estudiante.
    • Opción profesional. Desarrollar una aplicación que combine consultas SPARQL e inferencias sobre una ontología que resulte de ampliar el dominio del ejercicio anterior con más características.
    • Opción investigadora. Crear un pequeño artículo de investigación sobre un tema de interés para el alumno relacionado con la Web Semántica. El artículo tendrá un tamaño máximo de 5 folios y deberá seguir el formato de Springer-Verlag. El trabajo podrá complementarse con algún desarrollo o prototipo, que estará disponible en la página web del alumno. En este trabajo se valorará especialmente la originalidad de la propuesta.
