Asignaturas: Web Semántica y Nuevos Avances en Web Semántica

Asignaturas del Máster Oficial de Ingeniería Web de la Universidad de Oviedo.

Tarea a realizar para Daniel Gayo

Desarrollar una aplicación que cumpla, al menos, los siguientes requisitos:

  1. Obtendrá datos generados por usuarios mediante el Streaming API de Twitter en su versión pública (i.e. spritzer)
  2. Extraerá términos de dichos datos mediante el servicio web de OpenCalais en su versión para lengua inglesa
  3. Mostrará la lista de términos (extraídos mediante OpenCalais) más frecuentes en los tweets de la última hora (obtenidos mediante Twitter Streaming API).

Ampliaciones

  • Visualizar los términos como una nube de etiquetas en lugar de una lista.
  • Emplear, además de la frecuencia de aparición en el stream, la Entity Relevance Score proporcionada por OpenCalais para ajustar el "peso" de las entidades descubiertas.

Entregables

  • Documentación describiendo, de manera concisa, las características de las APIs utilizadas, las bibliotecas empleadas y los problemas encontrados (y solventados) durante la realización del ejercicio.
  • Código fuente completo.
  • Además, es obligatorio tener funcionando la aplicación en el servidor propio del alumno para su prueba.
