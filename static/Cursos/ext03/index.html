XML, Servicios Web y Web Semántica

Datos del curso

  • Dirección: Jose Emilio Labra Gayo
  • Fechas: 12 al 26 de Marzo de 2003
  • Lugar: Escuela Universitaria de Ingeniería Técnica Informática de Oviedo
  • Plazas: 25
  • Requisitos: Conocimientos de informática y programación
  • Tipo: Homologable por 4.5 créditos de libre configuración
  • Objetivos:

    Se espera que al finalizar el curso, el alumno sea capaz de:

    • Utilizar el lenguaje HTML y las hojas de estilo CSS para crear páginas Web fácilmente mantenibles
    • Utilizar el lenguaje XML para representar información concreta
    • Utilizar el lenguaje XSLT para transformar información representada en XML
    • Conocer vocabularios específicos basados en XML
    • Desarrollar programas mediante una arquitectura basada en servicios Web
    • Conocer el lenguaje RDF que facilita la creación de la Web semántica

Evaluacion

La evaluación del curso se desarrollará mediante la realización de una página Web que se construirá utilizando las tecnologías presentadas en el curso.

Se deben incluir los siguientes enlaces obligatorios:

  • Información. Enlace a información en HTML generada a partir de documentos XML. Opcionalmente, se puede incluir otras tecnologías dinámicas (servicios Web, acceso a bases de datos, etc.)
  • Código Fuente. Incluir el código fuente en un fichero ZIP comprimido. Añadir un fichero "readme.txt" que describa brevemente las tecnologías empleadas
  • Trabajo teórico. Realizar un pequeño trabajo (no más de 5 hojas) sobre alguna tecnología o vocabulario XML relacionado con el curso. Puede incluirse en formato PDF

La página debe finalizarse antes del 22 de Abril. A partir de dicho día comenzará el proceso de evaluación. Si la página no está accesible, la nota del alumno será "NO PRESENTADO".

Las notas podrán consultarse a través de Internet de forma individual en la dirección indicada en el apartado alumnos.
