Software a Instalar

Para poder ejecutar los ejemplos y ejercicios que se verán en el curso puede obtenerse el fichero cursoXML.zip.

Para que funcione corréctamente, descomprimir el fichero en un directorio en el que se tengan permisos de escritura.

Una vez descomprimido dicho fichero, debería quedar todo el software ya instalado. De todas formas, comprúebese que funciona correctamente.

Otra alternativa sería instalar directamente los originales que se indican en la siguiente lista siguiendo las instrucciones correspondientes.

  • Librería Jena. Se utilizarán algunas de las utilidades en línea de comandos. Comprobar que el CLASSPATH está inicializado corréctamente y comprobar, por ejemplo que la siguiente instrucción funciona: java jena.sparql
  • Isaviz. Este programa requiere también tener instalado GraphViz
  • Protègè.
  • Pellet.
